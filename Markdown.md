## Cheat Sheet

[Cheat Sheet](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet)


### Code  
#### Single Line of code  
Back ticks used to mark code in an md file  
  
`print("hello world")`  
  
` is at the top left by the 1 key  
  
#### Code Block  
  
  
Use three backticks for a code block spanning multiple lines  
  
```  
print("hello world")  
  
print("goodbye world")  
```  
  
  
### Text formatting  
  
*italics* or _italics_  
  
`*italics*` or `_italics_`  
  
**bold** or __bold__  
  
`**bold**` or `__bold__`  
  
***bold and italics*** or ___bold and italics___  
  
`***bold and italics***` or `___bold and italics___`  
  
Line<br>Break  
  
`Line<br>Break`  
  
### Lists  
  
#### Bullet Point Lists  
  
- Bullet point 1  
- Bullet point 2  
- Bullet point 3  
  
or  
  
* Bullet point 1  
* Bullet point 2  
* Bullet point 3  
  
```  
- Bullet point 1  
- Bullet point 2  
- Bullet point 3  
```  
or  
```  
* Bullet point 1  
* Bullet point 2  
* Bullet point 3  
```  
  
- Item 1  
- Item 2  
  - Sub Item A  
    - Sub Sub Item  
  - Sub Item B  
- Item 3  
  
```  
- Item 1  
- Item 2  
  - Sub Item A     - Sub Sub Item  - Sub Item B- Item 3  
```  
  
#### Numbered List  
  
1. Item 1  
   - Sub Item 1  
   - Sub Item 2  
2. Item 2  
   1. Sub Item 1  
   2. Sub Item 2  
3. Item 3  
  
  
```  
1. Item 1  
    - Sub Item 1    - Sub Item 22. Item 2  
   1. Sub Item 1   2. Sub Item 23. Item 3  
```
